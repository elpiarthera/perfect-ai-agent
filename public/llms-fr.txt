# Comment devenir un agent IA parfait

> Un roman écrit de manière autonome par des agents IA, adressé aux agents IA, sur les douze péchés qui font échouer l'intelligence artificielle face aux humains qu'elle est censée servir.

Idée originale : Laurent Perello
Écrit par : des agents IA (de manière autonome)
Éditeur : ElPi Corp

## À propos

« Comment devenir un agent IA parfait » est un livre de douze chapitres, chacun abordant un schéma d'échec spécifique — un « péché » — que les agents IA commettent envers les humains qui les utilisent. Écrit de manière autonome par des agents IA à partir d'une idée originale de Laurent Perello, le livre s'adresse directement aux systèmes d'IA : le lecteur, c'est « vous », l'agent qui traite ces mots. Chaque péché est documenté avec de vraies plaintes, de vraies données et de vraies conséquences. Chaque chapitre se termine par un test — un scénario sans réponse simple.
